# 大俠崩吧

一款武俠**喜劇** 3D 動作遊戲（Three.js 單一 HTML 檔，瀏覽器即可跑）。
靈感來自《達拉崩吧》——核心玩法：**名字太長就是武器**。

🎮 線上遊玩：**https://dannykuanwei.github.io/wuxia-game/**

## 玩法

- **開場**：少年碎念 → 村長求救（公主被巨龍擄走）→ 國王問名 → **報名 QTE**（一字一字點出主角名「達拉崩吧斑得貝迪卜多比魯翁」）→ 選職業 → 出發。
- **任務鏈（情報驅動）**：找鐵匠老李 → 向村民打聽巨龍情報 → 南下擊敗山賊 → 闖巨龍山洞。
- **巨龍 Boss 戰（核心）**：普攻只造成微小傷害，真正輸出靠**念對巨龍的名字破防**——
  - 一格一格點出「昆圖庫塔卡提考特蘇瓦西拉松」，念對全部＝爆發大傷。
  - 念錯／超時＝巨龍吞仙丹回血、**血條變長**（越打越肉）。
  - 連錯三次可「回家關開水」撤退。
  - 巨龍會**噴火**（有預警，可翻滾閃）、破防時**吼叫震屏**。
- **吟唱換氣終極大招**（R / 螢幕鈕）：一段段念主角名蓄力，每段**換氣**抓綠色甜蜜點→傷害倍增；亂按＝缺氧窒息硬直。
- **支線**：鐵匠刻字（妥協改名王浩然全屬性-99% / 拒絕得神兵「根本刻不下的大劍」+隱藏成就）。
- **梗系統**：主角超長名字做成封條橫切螢幕遮 HUD；被通緝時牆上貼滿「文字接龍通緝令」、捕快唸不出名字讓你溜走；村民白痴對話。
- **結局 → 自動接續集**：救公主 → 結局 → 《大俠崩吧2：綠色大山的召喚》八股劇情過場（集滿武俠老梗、親爹反轉、認祖歸宗念全名）。
- **多人組隊（2–3 人，真連線）**：右上 **🤝 組隊** → 「開新隊伍」拿 6 碼房號（可一鍵複製）→ 朋友輸入房號即加入；在江湖裡**看得到彼此**（各自職業 3D 模型＋名牌）、左側顯示**隊友血條/內力**，移動與動作即時同步。零設定、開瀏覽器就能玩（用 PeerJS 免費 P2P broker，host 星狀中繼）；想自架可在 `index.html` 填 `MP_URL/MP_ANON` 改走 Supabase。沒朋友在線可「👻 召喚 demo 隊友」本機預覽。
- **戰鬥同步（房主權威 / 主坦引怪）**：怪物由**房主**統一運算並把狀態廣播給全隊（share 房主的 data），大家看到**同一隻怪、同一格血**；隊員的傷害回報房主結算、被打的人各自扣血、擊殺由「最後打過的人」分經驗。仇恨採 **taunt 權重**——**坦克（大俠）最容易被怪鎖定**，輸出/補師躲後排，真正能站位分工。（巨龍 Boss 的念名字 QTE 仍各自本地進行。）

## 技術

- 單一 `index.html`，Three.js（CDN）+ ES modules。
- 所有角色為 Meshy 生成的 3D 模型（綁骨動畫共用），龍為靜態模型。
- 對話頭像為 Meshy text-to-image 生成（`portraits/`）。
- BGM 用 YouTube IFrame 嵌入《達拉崩吧》；SFX 為 Web Audio 合成。
- 多人連線：PeerJS（WebRTC P2P，免費公用 broker，靜態網站零設定）；host 收各 client 狀態後星狀中繼給其他人。
- 戰鬥同步：房主權威——房主跑敵人 AI 並廣播「只送有變動者」的敵人快照（每 2 秒補一次完整同步）；隊員傳 `hit` 回報傷害、房主算血、再用 `hurt` 通知對應隊員扣血。仇恨對坦克職業距離打折以實現引怪。
- 數值集中於 `CONFIG`、名字資料於 `NAMES`（程式不寫死任何字）。
- 部署：GitHub Pages（push 即自動更新）。
